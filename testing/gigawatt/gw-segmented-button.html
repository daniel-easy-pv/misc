Title

Content Switcher to be used for instances similar to a dropdown where the content wants to be more visible. It should only be used for 2-5 options. In instances where there are six or more options a dropdown selector should be used.


Icon content switchers should only be used in rare instances where the icons purposes are very clear and there is not enough space for words.
